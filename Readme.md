# EventEase (Proyecto Blazor)

Proyecto simplificado para cumplir los requisitos del curso Microsoft Front-End Developer.

## ✔ Funcionalidades

- Componente EventCard generado con Copilot.
- Enrutamiento entre páginas.
- Formulario de registro con validación.
- Gestión de estado con un servicio.
- Registro de asistencia.

## ✔ Cómo Copilot ayudó

- Generó la estructura inicial del componente EventCard.
- Sugirió el enrutamiento básico en App.razor.
- Propuso el uso de EditForm y DataAnnotations para validación.
- Ayudó a depurar errores de navegación.
- Recomendó simplificar el servicio de eventos para mejorar rendimiento.
